# GamDen 用户自主小程序申请与部署交互需求文档

**版本**：V1.0 | **状态**：待评审 | **关联文档**：《PRD-V1.0-邀请裂变系统》、《营销与引流渠道需求》


## 一、概述

### 1.1 文档定位

本文档定义了GamDen用户从 **“获得个人小程序资格”** 到 **“小程序成功部署上线”** 的完整交互流程、页面设计、状态管理和引导话术。

### 1.2 触发条件

用户邀请好友数量 **≥ 3人** 时，自动获得“个人专属小程序”的申请资格。

### 1.3 核心目标

| 目标 | 说明 |
|------|------|
| **降低门槛** | 让不懂技术的用户也能完成小程序的申请与部署 |
| **明确预期** | 提前告知用户所有成本（时间、金钱、材料），避免中途放弃 |
| **状态透明** | 每个阶段用户都能看到当前进度和下一步操作 |
| **引导回流** | 小程序部署成功后，持续引导用户回到GamDen App |

### 1.4 用户画像假设

GamDen的目标用户（数码爱好者、游戏玩家）具备以下特征：
- 大部分**不具备编程能力**，但对互联网产品有基本认知
- 对“拥有自己的小程序”有好奇心和社交炫耀动机
- 愿意为“独一无二的个人领地”付出一定的时间和少量金钱成本

## 四、后台状态管理

### 4.1 用户小程序状态字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `mini_program_status` | enum | `not_started` / `certifying` / `certified` / `deploying` / `reviewing` / `online` |
| `mini_program_appid` | string | 用户填写的AppID |
| `mini_program_name` | string | 用户自定义的小程序名称 |
| `certification_type` | enum | `individual` / `enterprise` |
| `certification_submitted_at` | timestamp | 用户提交认证的时间 |
| `appid_submitted_at` | timestamp | 用户提交AppID的时间 |
| `code_submitted_at` | timestamp | 代码提交审核的时间 |
| `online_at` | timestamp | 上线时间 |
| `mini_program_qr_code` | string | 小程序码图片URL |

### 4.2 状态流转规则

```
not_started ──(用户点击"开始申请")──→ certifying
certifying ──(用户确认已提交认证)──→ certified (或手动标记)
certified ──(用户提交AppID)──→ deploying
deploying ──(系统自动/用户确认提交审核)──→ reviewing
reviewing ──(用户确认已上线)──→ online
```

### 4.3 后台运营能力

| 能力 | 说明 |
|------|------|
| **查看所有用户的小程序状态** | 列表展示，支持按状态筛选 |
| **手动推进状态** | 当用户遇到困难时，运营可代为标记状态（需记录操作日志） |
| **发送提醒** | 向长时间停留在某一状态的用户发送推送提醒 |
| **数据统计** | 各状态用户数、转化漏斗（获得资格→开始申请→认证通过→部署→上线） |

## 七、技术实现要点

### 7.1 与主App的代码复用

根据之前确认的技术方案，个人小程序与GamDen App **共用同一套UniApp代码**，通过**条件编译**区分：
- App端：完整功能（地图、IM、集市、俱乐部）
- 小程序端：仅展示领地缩略图、邀请进度、引导下载

### 7.2 小程序代码的自动化构建

当用户完成认证并提交AppID后，系统应**自动构建**该用户的专属小程序：
1. 从GamDen主代码库拉取最新代码
2. 根据用户ID注入个性化配置（领地坐标、守护灵类型、自定义名称）
3. 调用微信CI工具链，上传代码至微信服务器
4. 通知用户“代码已提交审核”

> **注意**：微信小程序代码上传需配置**上传密钥**，需在微信公众平台「开发管理」中生成。

### 7.3 AppID绑定与校验

- 用户提交AppID后，系统应调用微信API校验该AppID的有效性
- 校验通过后，将AppID与用户账号绑定，存储在 `mini_program_appid` 字段
